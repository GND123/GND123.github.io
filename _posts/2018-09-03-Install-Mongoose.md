### MAC 기준
***
1. Node.js 설치 (<https://nodejs.org/ko/download/>)

2. 터미널에 아래 명령어를 입력해서 정상적으로 나오는지 확인한다.
~~~
node -v
~~~

3. mongoose 설치 터미널에 아래 명령어 입력
~~~
npm install mongoose
~~~

4. express 설치
~~~
npm install express
~~~
